1) PASSO: Clonar o repositório.. que será baixado na última versão

git clone https://github.com/bernafan/TrabalhoPOO

2) PASSO: Depois de uma alteração..

git add *

3) PASSO: Confirmar as alterações..

git commit -m "Comentario sobre o que foi alterado"

4) PASSO: Subir as alterações(Colocar o usario e senha do github)

git push origin

OBS: Sempre que for vizualizar/alterar a pasta, atualize-a antes, para evitar erros!!

git pull	

OBS2: Mudar de branch.. entre no diretorio do projeto, e digite:

git checkout nomeDaBranch
